The `supa` CLI helps you manage Supabase projects from the terminal. Sync schemas, manage configurations, and deploy changes across environments.

## Installation

```bash
npm install -g @supabase/cli
```

## Quick Start

Initialize a project:

```bash
supa init
```

Start development mode (watches for changes):

```bash
supa dev
```

Push changes to your project:

```bash
supa project push
```

## Authentication

Set your access token:

```bash
export SUPABASE_ACCESS_TOKEN=your-token
```

The CLI also loads tokens from `.env`, `supabase/.env`, and `.env.local` files.
